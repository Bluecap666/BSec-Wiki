# 扫描器

### 一、 综合型漏洞扫描器（功能全面）

这类具备强大的爬虫和漏洞库，能够识别并检测已知漏洞。

1. **Nessus**
   * **简介** ：全球最著名的商业漏洞扫描器之一，漏洞库极其庞大且更新及时。
   * **特点** ：能够自动识别中间件类型和版本，并基于其版本号匹配CVE漏洞。报告专业，准确性高。
   * **适用场景** ：企业级安全评估、合规性检查。
2. **OpenVAS**
   * **简介** ：Nessus的一个分支，现已发展成为功能强大的开源漏洞扫描器。
   * **特点** ：免费、开源，社区活跃，漏洞更新速度快。功能与Nessus类似。
   * **适用场景** ：预算有限的团队、安全研究人员和个人学习。
3. **Nexpose**
   * **简介** ：Rapid7公司出品的商业漏洞管理工具。
   * **特点** ：不仅扫描漏洞，还提供风险评分和修复优先级建议，与Metasploit框架无缝集成。
   * **适用场景** ：需要与渗透测试流程紧密结合的企业。
4. **AWVS**
   * **简介** ：Acunetix出品的商业Web漏洞扫描器，在Web应用安全领域非常知名。
   * **特点** ：对Web服务器（如Nginx， Apache， IIS）及其配置有深入的检测能力，也能检测到这些组件的相关漏洞。
   * **适用场景** ：专注于Web应用安全的团队。
