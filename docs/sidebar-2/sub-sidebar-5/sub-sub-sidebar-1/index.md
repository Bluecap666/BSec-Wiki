# 漏洞库

### 一、 漏洞情报与PoC集合

这些不是直接的工具，而是寻找利用代码的宝库。

| 工具/平台名称           | 类型        | 项目地址                                                                            | 核心特点                                              |
| ----------------------- | ----------- | ----------------------------------------------------------------------------------- | ----------------------------------------------------- |
| **PoC-in-GitHub** | PoC集合     | [https://github.com/nomi-sec/PoC-in-GitHub](https://github.com/nomi-sec/PoC-in-GitHub) | **强烈推荐** 收集GitHub上各种CVE漏洞的PoC代码。 |
| **CVE**           | PoC集合     | [https://cve.mitre.org/](https://cve.mitre.org/)                                       | 官方的CVE数据库，可据此在GitHub搜索。                 |
| **ExploitDB**     | PoC/Exp集合 | [https://www.exploit-db.com/](https://www.exploit-db.com/)                             | 著名的漏洞利用代码库，内容全面。                      |
